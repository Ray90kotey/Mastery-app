🧠 MASTERY SCORING ENGINE (INTELLIGENT)

Design a structured algorithm that considers:

Assessment performance

Assessment type weight

Recency of performance

Consistency trend

📊 Assessment Weights

Use weighted system:

Classwork → 10%

Quiz → 20%

Test → 30%

Project → 40%

(Make configurable later.)

🧮 Calculation Logic
Step 1: Assessment Percentage

percentage = (student_score / total_score) × 100

Step 2: Apply Assessment Type Weight

weighted_score = percentage × weight

Step 3: Apply Recency Multiplier

Most recent assessments → × 1.2

Mid-term → × 1.0

Older → × 0.8

This rewards improvement over time.

Step 4: Lesson Mastery

lesson_mastery = sum(weighted_scores) / sum(weights)

Step 5: Outcome Mastery

Aggregate all assessments linked to that outcome.

Step 6: Overall Mastery

Weighted average of all outcome mastery scores.
